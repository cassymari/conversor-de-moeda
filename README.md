# 💱 Conversor de Moedas

Aplicação web desenvolvida para converter valores entre diferentes moedas de forma rápida e prática.

O projeto foi criado com o objetivo de praticar **JavaScript, manipulação do DOM e lógica de programação**, além de trabalhar com conversão de valores e interação com o usuário.

# ✨ Funcionalidades

* Conversão entre diferentes moedas
* Atualização automática do valor convertido
* Interface simples e intuitiva
* Exibição do valor convertido em tempo real

---

# ⚙️ Como usar

1. Digite o valor que deseja converter
2. Escolha a moeda de origem
3. Escolha a moeda de destino
4. O valor convertido será exibido automaticamente
